# Total Counter Mail

RICOH 複合機（Web Image Monitor）からトータルカウンターを取得し、月1回メールで送信する Windows アプリです。

## 動作確認済み機種

- **imagio MP W2401**（本番利用中）

アプリ画面上には **「動作確認済み: MP W2401」** のみ表示されます。候補機種の一覧や詳細案内はアプリ内には出しません。

その他の候補機種・接続テスト手順は、配布フォルダまたは本リポジトリの [対応機種候補一覧.txt](対応機種候補一覧.txt) を参照してください。  
一覧に載っていても、ゲスト CGI の有無や HTML 形式の違いにより動作しない場合があります。

## 使い方

### 1. 起動

`Total Counter Mail.exe` をダブルクリックして起動します。  
設定は exe と同じフォルダの `config.ini` に保存されます（暗号化）。

### 2. 接続設定タブ

| 項目 | 内容 |
|------|------|
| 複合機IPアドレス | 例: `192.168.11.111` |
| SMTP / POP3 | メールサーバー情報 |
| 送信元・送信先 | To / CC（任意2件） |

1. 複合機 IP を入力し **接続** でカウンター取得を確認
2. メール設定を入力
3. **保存** で設定を書き込み
4. **テスト送信** でメール送信を確認

### 3. メール文面タブ

件名・前置き文・カウンター見出し・日時ラベル・締め文を編集できます。  
本文の各項目の間には空行が入ります。

### 4. 自動送信タブ

| 項目 | 内容 |
|------|------|
| 毎月何日 | 送信予定日（1〜31） |
| 何時 | 送信時刻（HH:MM） |

- 指定日が **土日** の場合、送信は **翌月曜日** に繰り越されます
- PC が予定時刻にオフでも、**起動後に追送** されます
- 自動送信の登録には [利用規約](利用規約.txt) への同意が必要です
- **自動送信を登録** で Windows タスクスケジューラに登録されます（毎日実行、アプリ側で月1回のみ送信）
- exe を移動・更新した場合、起動時にタスクが自動同期されます

### 5. 手動送信

**今すぐ送信** でその場でカウンター取得＋メール送信を行えます（規約同意不要）。

## ファイル構成（配布フォルダ）

```
Total Counter Mail.exe   … アプリ本体
config.ini               … 設定（初回保存後に作成、暗号化）
log/                     … 送信ログ・送信履歴
利用規約.txt             … 自動送信機能の利用規約
対応機種候補一覧.txt     … 対応機種の参考リスト（アプリ内では案内しない）
```

## 利用規約（自動送信機能）

自動送信機能の利用には [利用規約.txt](利用規約.txt)（版 1.3）への同意が必要です。  
手動送信・テスト送信・設定保存には規約は適用されません。

主な内容:

- 設定・ログは利用者 PC 上のみに保存（提供者への送信なし）
- 現状有姿での提供（動作保証・損害補償なし）
- **利用可能期間: 2026年12月31日まで**（2027年以降は自動送信の新規登録・実行不可）

全文は [利用規約.txt](利用規約.txt) を参照してください。
