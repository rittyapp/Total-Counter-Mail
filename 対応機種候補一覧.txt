================================================================================
【重要】この一覧の読み方
================================================================================

■ 実際に動作確認済み
  - imagio MP W2401 のみ（本プロジェクトで接続テスト・送信実績あり）

■ それ以外は「対応しそう」な候補
  公式に「この機種だけ対応」とは言えません。
  あくまで、Web Image Monitor とカウンター CGI の仕様が近いと推測したものです。

■ アプリが機種対応を判定する条件（技術メモ）
  1. 次の URL に HTTP でアクセスできること（ログイン不要のゲストパス）
     http://{複合機IP}/web/guest/ja/websys/status/getUnificationCounter.cgi
  2. 返ってくる HTML に「トータル」カウンターが含まれること
  3. 上記 HTML の表形式が、アプリが想定している形式と一致すること

  ※ Web Image Monitor 自体はあるが、ゲスト公開がオフ・HTTPS 強制・
    画面形式が違う場合は「非対応」扱いになります。

■ 参考にした公開情報
  - リコー FAQ 2185
    「Web Image Monitor でカウンターを確認する方法」
    対象例: RICOH IM C/IM/MP C/MP、imagio MP C/MP、レーザープリンター 等
    https://faq.ricoh.jp/app/answers/detail/a_id/2185
  - リコー 販売終了品一覧（imagio MP 系の型番整理）
    https://www.ricoh.co.jp/products/discontinued/multi-function-printer

================================================================================
【E】非対応・要確認になりやすいパターン（機種名ではなく条件）
================================================================================

  × Web Image Monitor が無効
  × ゲストアクセスが禁止されている
  × HTTPS のみで HTTP が使えない（アプリは現在 http:// 固定）
  × IP 直打ちできない（認証プロキシ配下など）
  × カウンター画面に「トータル」が無い（ユーザー別のみ等）
  × HTML 表レイアウトが異なり数値抽出に失敗する
  × アナログ複写機・ジアゾ複写機などデジタル MFP 以外

================================================================================
【実機で試すときの手順（1分チェック）】
================================================================================

  1. ブラウザで次を開く
     http://{複合機IP}/web/guest/ja/websys/status/getUnificationCounter.cgi

  2. ページが開き、「トータル」に数値が見えれば候補成立

  3. アプリの「接続」ボタンでも同じ IP を試す

  4. 失敗したら
     - IP / 電源 / 同一 LAN を確認
     - 404 → CGI 非対応の可能性大
     - 200 だが数値なし → 画面形式不一致の可能性

================================================================================
【アプリ側で将来やると良いこと（メモ・未実装）】
================================================================================

  - 機種名をアプリ表示から外す（固有名詞回避）
  - 「対応確認済み」「要テスト」の公式リストをユーザーが編集できるようにする
  - HTTPS 対応（証明書エラー回避）
  - HTML 形式のゆらぎ吸収（機種別パターン追加）

【まとめ】
================================================================================

  今すぐ安心して使えるのは imagio MP W2401 のみ。

  次に試す価値が高いのは imagio MP W 系（W3601 / W5100 / W7140）と
  RICOH MP W 系（W4002 / W7100 / W8140）。

  事務系 imagio MP / imagio Neo / RICOH IM・MP は Web Image Monitor 対応の
  可能性があるが、ゲスト CGI と HTML 形式の確認が必要。

  最終判断は必ず実機の URL 確認またはアプリの接続テストで行うこと。
